KT2 Remote Control

Panel estático para controlar el Kungfu Turtle KT2 desde el navegador.

Conexión

Listo.

El panel usa llamadas relativas al ejecutarse desde el propio servidor del KT2.

Luces

Sonido

Movimiento

Flechas para caminar y girar. También puedes usar el teclado: flechas, espacio y B.

Actions

Botones compactos con acciones comunes del módulo actions.

Python rápido

Log

Atajos de teclado

↑ avanzar
↓ retroceder
← girar izquierda
→ girar derecha
espacio = parar
B = bark
F = back flip
R = reset postura

Notas

Cómo están hechas las llamadas
La página usa el endpoint /py para ejecutar código Python embebido, por ejemplo q.play(actions.back_flip(q), dly=0.1) o car.led.on(0xff0000).
Compatibilidad
Está preparada para el KT2 con las APIs no oficiales que has encontrado. Algunas acciones pueden variar según firmware.
